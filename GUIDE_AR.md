# 🚀 دليل نظام حجز المواعيد المتكامل (SaaS Booking Platform)

مرحباً بك! هذا الملف يشرح هيكلية المشروع، المهارات المستخدمة، وكيفية عمل النظام ككل.

## 🗺️ خريطة الموقع (Site Map)

### 1️⃣ الموقع الرئيسي (Landing Page)
- **الرئيسية (`/`)**: لشرح فكرة المنصة وجذب أصحاب الأعمال.
- **المميزات (`/features`)**: تفاصيل ما يقدمه النظام.
- **الأسعار (`/pricing`)**: خطط الاشتراك الشهرية.

### 2️⃣ واجهة صاحب العمل (SaaS Owner)
- **بداية العمل (`/onboarding`)**: واجهة احترافية لإعداد بيانات البزنس عند التسجيل لأول مرة.
- **لوحة التحكم (`/dashboard/overview`)**: ملخص شامل للأرباح والحجوزات.
- **إدارة الخدمات (`/dashboard/services`)**: إضافة وتعديل وحذف الخدمات والأسعار.
- **إدارة المواعيد (`/dashboard/bookings`)**: متابعة حجوزات الزبائن وتأكيدها.

### 3️⃣ واجهة الزبون (Public Customer Page)
- **صفحة الحجز (`/[slug]`)**: صفحة عامة واحترافية خاصة بكل بزنس (مثلاً: `/ahmad-barber`) حيث يمكن للزبائن الحجز مباشرة.

---

## 🏗️ الأبنية التقنية (Tech Stack)
تم بناء المشروع بأحدث التقنيات لضمان السرعة والأمان:
- **Next.js 15 (App Router)**: للسرعة الفائقة وتحسين محركات البحث (SEO).
- **Prisma + PostgreSQL**: قاعدة بيانات قوية تدعم تعدد المستأجرين (Multi-tenancy).
- **JWT + Cookies**: نظام حماية وتشفير متطور لإدارة الجلسات والأدوار (Roles).
- **Tailwind CSS + Framer Motion**: لتصميم عصري (Premium) وحركات سلسة (Micro-animations).
- **Lucide & React Icons**: أيقونات احترافية.

---

## 🛠️ المهارات الجديدة التي تم تطبيقها
1. **Multi-tenancy Layout**: بناء نظام واحد يخدم مئات الشركات بشكل منفصل.
2. **Dynamic Routing**: إنشاء صفحات ديناميكية بناءً على رابط البزنس.
3. **Complex Prisma Transactions**: إدارة إنشاء الحساب والعمل التجاري في خطوة واحدة بأمان.
4. **Middleware Protection**: حماية الروابط بناءً على صلاحية المستخدم (Admin, Owner, Staff).
5. **Floating Label Inputs**: بناء عناصر UI مخصصة تزيد من جمالية التجربة.

## 👑 دورك كـ Super Admin
بصفتك المالك لهذه المنصة، لديك لوحة تحكم خاصة (`/admin`) تتيح لك:
- متابعة عدد المستخدمين الجدد.
- رؤية الشركات المنضمة مؤخراً.
- مراقبة حالة النظام والأرباح الإجمالية من الاشتراكات.
